专有名词解释

股市中有一些约定俗成的词语来表达一些特定的含义,这就是股市术语,也叫股票术语,下面就介绍一些股市常用的术语：点击进入>>

  • 牛市
    • 也称多头市场，指市场行情普遍看涨，延续时间较长的大升市📈。
  • 熊市
    • 也称空头市场，指行情普遍看淡，延续时间相对较长的大跌市📉。
  • 牛皮市
    • 是指在一段交易日内，证券价格上升和下降的幅度很小，价格变化不大。在牛皮市上往往成交量也很小，牛皮市是一种买卖双方在力量均衡时的价格行市表现。
  • 多头
    • 是指对股市前景看好，预计股价会上涨而逢低买进股票的投资者。
  • 空头
    • 是指对股市前景看坏，预计股价会下跌，而逢高卖出股票的投资者。
  • 利多
    • 又叫利好，是指刺激股价上涨的信息，如上市公司经营业绩好转、银行利率降低以及其他政治、经济、军事、外交等方面对股价上涨有利的信息。
  • 利空
    • 对于空头有利，致使股价下跌的各种因素和消息，如上市公司经营业绩恶化、银行利率调高、经济衰退、通货膨胀、天灾人祸以及其他政治、经济、军事、外交等方面促使股价下跌的不利消息称为利空。
  • 做多
    • 就是做多头。多头对市场判断是上涨，就会立即买入股票，所以做多就是买入股票。
  • 做空
    • 就是做空头。空头对市场判断是下跌，就会立即卖出股票，所以做空就是卖出股票。
  • 诱多
    • 指主力、庄家有意制造股价上涨的假象，诱使投资者买入，结果股价不涨反跌，让跟进做多的投资者套牢的一种市场行为。
  • 诱空
    • 指主力、庄家有意制造股价下跌的假象，诱使投资者卖出，结果股价不跌反涨，让卖出的投资者踏空的一种市场行为。
  • 踏空
    • 股价上涨，而原来的抛出股票的投资者，因某种因素没有及时买进，这种现象称为踏空。
  • 长空
    • 是对股价前景看坏，借来股票（融券）卖出，或卖出股指期货，等股票或股指期货长期下跌后再买回来获得利润的行为。
  • 短空
    • 是指对股价短期内看跌，借来股票（融券）卖出，待股价短期回落买回来获利的行为。
  • 长多
    • 是指对股价远期看好，认为股价长期会不断上涨，因而买进股票长期持有，等股价上涨相当长时间后再卖出，赚取差价收益的行为。
  • 短多
    • 是指对股价短期内看好，买进股票，如果股价略有上涨即卖出的行为。
  • 死多
    • 是指看好股市前景，买进股票后，如果股价下跌，宁愿放上几年也绝不脱手。
  • 翻多
    • 空头的投资者观念改变，开始看好后市行情，不但将做空的股票补回来，还买进股票。
  • 翻空
    • 原来是多头的投资者，看坏行情，把做多的股票卖掉。
  • 多杀多
    • 市场普遍认为当天股价将上涨，于是买入股票的投资者特别多，然而股价却没有大幅度上涨，等交易快结束时，反而竞相卖出，造成收盘价大幅度下跌的情况。
  • 轧空
    • 市场普遍认为当天股价将下跌，于是都卖出股票，然而股价并未大幅度下跌，无法低价买进，收盘前只好竞相补进，反而是收盘价大幅度升高的情况。
  • 空仓
    • 指投资者将所持有的股票全部抛出，手中持有现金而无股票的状态。
  • 建仓
    • 指投资者判断股价将要上涨而开始买进股票的投资行为。
  • 满仓
    • 是指投资者将资金全部买入了股票而手中已没有现金的状态。
  • 斩仓
    • 又称割肉、停损，是指在买入股票后，股票下跌，投资者为避免损失扩大而低价卖出股票的投资行为。
  • 减仓
    • 减仓就是卖出手中持有的部分股票，减少所拥有股票的数量。
  • 加仓
    • 在原来有股票的基础上，因持续看好该只股票，而在该股票上涨的过程中继续追加买入的行为。
  • 重仓
    • 投资者买入股票所用资金比例占总资金的2/3以上。
  • 轻仓
    • 投资者买入的股票所用资金比例占总资金的1/3以下。
  • 套牢
    • 投资者预计股价将上涨，但在买进后股价却一直呈下跌趋势，若卖出所持股票，肯定形成亏损；若继续持有，市场前景又不确定，从而被动地陷入一种左右为难的境地，这种现象称为套牢。
  • 补仓
    • 是股票被套牢后的一种被动应变策略，即以更低的价格购买该股票，使单位成本下降，以期在补仓之后反弹抛出，将补仓所买回的股票所专区的利润弥补高价位买入股票的损失。
  • 仓位
    • 仓位是指投资人实际投资和实有投资资金的比例。例如投资者总的投资金额为10万元，现在用了5万元买入了股票，那么该投资者的仓位就是50%。
  • 庄家
    • 这个词是从赌博中借用过来的，就是指在股市上，通过各种手段影响甚至操纵股价，以获得暴利的个人和机构。
  • 震仓
    • 指庄家在拉升过程中使股价震荡，清洗筹码的过程，目的是尽量把心态不坚定的短线跟风盘甩掉。
  • 追涨
    • 就是当股票开始涨起来时，不管价位是多少都买入股票的投资行为。
  • 杀跌
    • 就是在股市下跌的时候，不管当初股票买入的价格是多少，都立刻卖出，以求避免更大的损失，这种行为叫杀跌。
  • 止盈
    • 指股票升势没有发生变化，为了确保胜利果实，而在预先位置的目标价左右，提前卖出股票、锁定利润的一种行为。
  • 抄底
    • 在股票价格下跌过程中，估计快要到底部，价格要从跌转为升的时候，就果断买入，这就是所谓的抄底。抄底能否成功，在于对“底”的判断是否正确，如果价格继续下跌，就是抄错了。
  • 逃顶
    • 在股票价格上涨过程中，估计快要到顶部，价格要从涨转为跌的时候，就果断卖出，就是所谓的逃顶。
  • 盘整
    • 股价经过一段快速上升或下降后，遭遇阻力或支撑而呈小幅涨跌变动的情况。
  • 反弹
    • 是指在下跌的行情中，股价受到支撑暂时回升的现象，反弹后恢复下跌趋势。
  • 回档
    • 是指股价上升过程中，因上涨过快而暂时回跌的现象。
  • 坐庄
    • 指庄家以获利为目的，对某只股票买入卖出的行为。
  • 吸筹
    • 是指在股市中庄家、主力或大户介入某一只股时，不断买入股票（建仓或加仓）的行为。
  • 对敲
    • 是指庄家或大的机构投资者为了达到操纵股价的目的，蓄意串通，按照事先约定的方式或价格进行交易或互为买卖的行为。
  • 打压
    • 指用非常规的方法，将股价大幅压低，然后又大量买进以牟取暴利。
  • 洗盘
    • 庄家为达炒作目的，必须于途中让低价买进，意志不坚的散户抛出股票，以减轻上档压力，同时让持股者的平均价位升高，以利于施行坐庄的手段，达到牟取暴利的目的。洗盘动作可以出现在庄家任何一个区域内，基本目的无非是为了清理市场多余的浮动筹码，抬高市场整体持仓成本。
  • 出货
    • 通常指庄家在高价时，不动声色的卖出股票。
  • 散户
    • 就是买卖股票数量很少的小额投资者。
  • 大户
    • 指那些资金实力雄厚，投资额巨大，交易量惊人，能够左右行情，控制市况的投资者。
  • 中户
    • 指投资额较大的投资人。
  • 机构
    • 具有证券自营业务资格的证券经营机构，符合国家有关政策法规的投资管理基金。
  • 坐轿
    • 预测股价将涨，抢在众人前以低价先行买进，待众多散户跟进、股价节节升高后，卖出获利的现象。
  • 主力
    • 是持股数较多的机构或大户。每只股票都存在主力，但是不一定都是庄家，庄家可以操控一只股票的价格，而主力只能短期影响股价的波动。
  • 抬轿
    • 在别人早已买进后才醒悟，也跟着买进，结果是把股价抬高让他人获利，而自己买进的股价已非低价，无利可图的现象。
  • 筹码
    • 指投资人手中持有的一定数量的股票。
  • 抢帽子
    • 指当天先低价买进股票，然后高价再卖出早先买进的相同种类、相同数量的股票；或当天先卖出早先买进的股票，然后以低价买进相同种类、相同数量的股票，以求赚取差价利益。
  • 跳水
    • 指股价迅速下滑，幅度很大。
  • 多头陷阱
    • 指为多头设置的陷阱，通常发生在指数或股价屡创新高，随后迅速下滑跌破以前的支撑位，结果使在高位买进的投资者严重被套。
  • 空头陷阱
    • 指为空头设置的陷阱，通常出现在指数或股价从高位区以高成交量跌至一个新的低点，并造成向下突破的假象，恐慌性抛盘涌出后迅速回升至原先的密集成交区，并向上突破原压力线，使在低点卖出者踏空。
  • 护盘
    • 股市低迷、人气不足时，机构投资大户大量购进股票，防止股价继续下滑的行为。
  • 跳空
    • 股价受利多或利空影响后，出现较大幅度上下跳动的现象。
  • 开盘价
    • 指每天成交中最后的一笔成交的股票价格。
  • 收盘价
    • 指每天成交中最后一笔成交的股票价格。
  • 最高价
    • 指当天股票成交的各种不同价格中最高的成交价格。
  • 最低价
    • 指当天成交的不同价格中最低的成交价格。
  • 成交量
    • 指股票成交的数量。
  • 放量
    • 是指成交量比之前有明显放大。例如，某只股票昨天全天的成交量是1.5亿股，今天忽然变成2.5亿股，这就是放量。
  • 缩量
    • 与放量相反，是指成交量比之前有明显的减少。
  • 天量
    • 表示某只股票或整个市场当天出现的巨大的交易量，远远超过平时的成交量。
  • 天价
    • 指股票的价格达到一个较高的水平，一般表示股票已经处于上升的末期，离未来的下跌为时不远。
  • 热门股
    • 是指交易量大、流动性强、价格变动幅度大的股票。
  • 冷门股
    • 是指交易量小，流动性差甚至没有交易，价格变动小的股票。
  • 白马股
    • 一般是指其有关的信息已经公开，业绩优良，具高成长性、低风险的特点，因而具备较高的投资价值，往往为投资者所看好。
  • 黑马股
    • 是指脱离过去的价位而在短期内大幅上涨的股票。黑马股可遇而不可求，如果被大家都看好的股票就很难成为黑马股了。
  • 龙头股
    • 在股票市场的炒作中，对同行业板块的其他股票具有影响力和号召力的股票，他的涨跌往往对其他同行业板块股票的涨跌起引导和示范作用。
  • 阴跌
    • 指股价进一步退两步，缓慢下滑，如阴雨连绵，长期缓慢下跌不止。
  • 换手率
    • 也成周转率，指在一定时间内股票转手买卖的频率，是反映股票流动性强弱的指标之一。换手率 = 某一时间段内的成交量 / 流通股数 X 100% 。
  • 现手
    • 已经成交的最新一笔买卖的手数。
  • 平开
    • 股票的当日开盘价与前一交易日收盘价持平的情况称为平开。
  • 低开
    • 股票的当日收盘价低于前一交易日收盘价的情况称为低开。
  • 高开
    • 股票的当日开盘价高于前一交易日收盘价的情况称为高开。
  • 内盘
    • 就是股票在买入价成交，成交价为买入价，说明抛盘比较踊跃。
  • 外盘
    • 就是股票在卖出价成交，成交价为卖出价，说明买盘比较积极。
  • 均价
    • 就是某一时刻买卖股票的平均价格。
  • 浮筹
    • 指散户手中的股票，流动性强。这部分股票持有者短线投机心态较强，操作时每拉升到一个关键的技术位，他们手中的股票就急着变现。
  • 市盈率
    • 在一个考察期（通常为12个月的时间）内，股票的价格和每股收益的比例。投资者通常利用该比例值估量某股票的投资价值，或者用该指标在不同公司的股票之间进行比较。
  • 含权
    • 凡是有股利的股票，尚未除权，即为含权。
  • 回购
    • 是指上市公司利用现金等方式，从股票市场购回本公司发行在外的一定数额股票的行为。
  • 技术分析
    • 是指以市场行为为研究对象，以判断市场趋势并跟随趋势的周期性变化来进行股票及一切金融衍生物交易决策的方法的总和。技术分析认为市场行为包容消化一切。
  • 基本面分析
    • 一般是指对宏观经济面、公司主营业务所处行业、公司业务、同行业竞争水平和公司内部管理水平等多方面的分析。
  • 量比
    • 是衡量相对成交量的指标。它是指股市开市后平均每分钟的成交量与过去5个交易日平均每分钟成交量之比。
  • 每股净资产
    • 指股东权益与总股数的比率。其计算公式为：每股净资产 = 股东权益 / 总股数。这一指标反映每股股票所拥有的资产现值。每股净资产越高，股东拥有的资产现值越多；每股净资产越少，股东拥有的资产现值越少。通常每股净资产越高越好。
  • 每股收益
    • 又称每股税后利润或每股盈余，指税后利润与股本总数的比率。它是测定股票投资价值的重要指标之一，是分析每股价值的一个基础性指标，是综合反映公司获利能力的重要指标。
  • 趋势
    • 就是股票价格市场运动的方向。趋势的方向有3个：上升方向，下降方向和水平方向。趋势的类型有主要趋势、次要趋势和短暂趋势3种。
  • 市净率
    • 是每股股价与每股净资产的比率。一般来说市净率较低的股票，投资价值较高，相反，则投资价值较低。
  • 探底
    • “底”就是大盘或个股停止下跌的一段区间。股价下行后，有大量的承接盘使股价回升，这个下行过程就是探底，目的在于探明该股的底部哪个价位有大承接盘。探底不成功后，继续下探，直到某个底部价位出现大量承接盘使其难再下行时，称为探明底部 。底部探明后，股价就要反弹回升。
  • 填权
    • 在除权（除息）后的一段时间里，如果多数人对该股看好买入，导致股票上涨，使股票交易的市价高于除权（除息）基准价，这个行情称为填权。比如一只股票是10元，现在10股送10股，送股后的除权价变成了5元，如果该股从5元开始上涨就称为填权，如果一直上涨到10元就称为填满权。
  • 停牌
    • 股票由于某种消息或进行某种活动引起股价的连续上涨或下跌，由证券交易所暂停其在股票市场上进行交易，待情况澄清或企业恢复正常后，再复牌在交易所挂牌交易。
  • 退市
    • 是上市公司由于未满足交易所有关财务等其他上市标准而主动或被动终止上市的情形，既由一家上市公司变为非上市公司。
  • 委比
    • 是衡量某一时段买卖盘相对强度的指标，它的计算公式为：委比 = （委买手数 - 委卖手数）/ （委买手数 + 委卖手数）X 100%。委比的取值自 -100 到+100，+100表示全部的委托均是买盘，涨停的股票的委比一般是100，而跌停是-100.当委比为0时，表示是买入（托单）和卖出（压单）的数量相等，即委买：委卖 = 5：5。
  • 支撑线
    • 支撑线又称为抵抗线，当股价跌到某个价位附近时，股价停止下跌，甚至有可能回升，这是因为多方在此积极买入造成的。过这个价位的一条水平线便构成了支撑线，支撑线起阻止股价继续下跌的作用。
  • 阻力线
    • 股价上涨到达某一价位附近，股价停止上扬，甚至回跌。那么这个价位所构成的一条水平线便称为阻力线。形成阻力线的原因是空方在此大量卖出股票所致。阻力线起到了阻止股价继续上涨的作用。
  • 技术指标
    • 泛指通过数学公式计算得出的股票数据集合。目前，证券市场上的各种技术指标非常多，例如，相对强弱指标（RSI）、随机指标（KDJ）、趋向指标（DMI）、平滑异同平均线（MACD）等。
  • 金叉
    • 是指上升的中短期指标曲线由下而上穿过长期指标曲线，表示股价将继续上涨，行情看好。
  • 死叉
    • 是指上升的中短期指标由上而下穿过长期指标曲线，表示股价将继续下跌，行情看坏。
  • 背离
    • 是指技术指标曲线的运动方向与股票价格的运行方向的不一致，说明股价的变化没有得到指标的支持。背离分为顶部背离和底部背离。
  • 多头排列
    • 均线的排列顺序从上到下依次为短期均线、中期均线、长期均线，这说明过去买进的成本很低，做短线的、中线的、长线的都有利润，市场一片向上。多头排列代表多方（买方）力量强大，后市将由多方主导行情，此时是中线进场的机会。
  • 空头排列
    • 均线的排列顺序从上到下依次为长期均线、中期均线、短期均线，这说明过去买进的成本都比现在高，做短线、中线、长线的此时都在亏损，市场一片看坏，空头排列代表空方（卖方）力量强大，后市将由空方主导行情，此时是中线离场的机会。

  • 注册制
